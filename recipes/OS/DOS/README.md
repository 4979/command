# DOS [Disk Operating System]
**DOS** ([/d…ís/](https://en.wikipedia.org/wiki/Help:IPA_for_English#Key)), short for "Disk Operating System",[1] is an acronym for several closely related operating systems that dominated the IBM PC compatible market between 1981 and 1995, or until about 2000 if one includes the partially DOS-based Microsoft Windows versions 95, 98, and Millennium Edition.
